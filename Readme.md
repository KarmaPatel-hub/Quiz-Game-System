# Quiz Game System 🎮


## 📌 Project Overview

This project asks multiple-choice questions to the user, checks answers, and calculates the final score.

It is designed to demonstrate basic C++ programming concepts such as:

* Class and Object
* Functions
* Conditional Statements
* Variables
* User Input / Output

## 🎯 Objectives

* To build a simple quiz application in C++
* To understand OOP concepts through a practical project
* To improve logical thinking and coding structure

## ⚙️ Features

* 5 multiple-choice questions
* Instant answer checking
* Score calculation
* Final result display

## 🛠️ Technologies Used

* C++
* Console Application
* Visual Studio Code

## 📂 Project Structure

Quiz-Game-System/
│── main.cpp
│── README.md

## ▶️ How to Run

1. Open project in Visual Studio Code
2. Compile the code
3. Run the program
4. Answer quiz questions
